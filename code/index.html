Software Resources

  • Our Github repository has some examples and software and data resources for assignments
  • Apache Jena and Fuseki are two excellent open source packages for creating and using RDF triple stores
  • Protege is the most popular application for creating, editing and viewing semantic web ontologies and data
  • Pubby is a linked data frontend for SPARQL endpoints.
  • D2RQ is a system for accessing relational databases as Virtual RDF Graphs. An altnative and more recent system is ontop.
  • Meta SEO Inspector is a Chrome browser extension that shows meta data found ona web page, including Schema.org markup
